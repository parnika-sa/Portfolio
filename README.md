# My Portfolio

This project is a personal portfolio website built with React, Vite, and Tailwind CSS.

## Project info

This project showcases my work and skills.

## Getting Started

To run this project locally:

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Deployment

To deploy to Vercel:

1.  Push this repository to GitHub.
2.  Import the project in Vercel.
3.  Vercel will automatically detect the Vite build settings.
